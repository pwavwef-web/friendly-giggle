# 📁 Project Work — Dissertation Workspace

> TMG499 | University of Cape Coast | Level 400 | 2025/2026 Second Semester

This folder is the central workspace for the **TMG499 Dissertation (Project Work)**. All research materials, drafts, data, and final submission documents are organised here.

---

## Subfolder Guide

| Subfolder | Purpose |
|-----------|---------|
| [`proposal/`](./proposal/) | Research proposal drafts and final approved proposal |
| [`literature-review/`](./literature-review/) | Annotated bibliographies, reading notes, thematic summaries |
| [`methodology/`](./methodology/) | Research design, methods justification, sampling framework |
| [`data-collection/`](./data-collection/) | Survey instruments, interview guides, raw data files |
| [`data-analysis/`](./data-analysis/) | Analysis scripts, outputs, tables, and charts |
| [`drafts/`](./drafts/) | Versioned chapter drafts (never delete old versions) |
| [`supervisor-meetings/`](./supervisor-meetings/) | Meeting agendas, notes, feedback, and action items |
| [`final-submission/`](./final-submission/) | Formatted final dissertation ready for submission |
| [`references/`](./references/) | Zotero library exports, `.bib` files, bibliography |
